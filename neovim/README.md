## [`neovim`](https://neovim.io/)

### Install notes

Once installed, you can type `restow-dotfiles` to update stowed configs

### Notable shortcuts

- Unite Buffer List : mapped to C-v C-v
- [trailing-whitespace] : :FixWhitespace
- [nerdcommenter]       : Comment: ,cs - Toggle: ,c<space>
